About


Project Origin

This website is an assignment for a graduate level course, IS 601 - Web Systems Development. This course is offered by Professor Keith Williams at New Jersey Institute of Technology (NJIT).


Project Purpose

The purpose of this website is to learn the global network that we use quite often, to learn how the Internet works. We have tried to explain some of the basic concepts based on the modules shared such as Domain names and ICANN, IP addresses, DNS, Https and Http protocol and W3c, HTML and CSS basics. We hope you find this webpage useful!
